Detailed Description

The RocketData struct stores all data that is needed by more than one system/thread of the Rocket.

holds all information about the rocket, sensors, and controllers

Note
Normally, this would be considered a poor decision. However, the fact that all this data is here makes it easier to debug since all this data can be logged (and thus used when debugging).
